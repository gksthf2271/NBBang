# NBBang

## 개발 목적
더치페이 및 그룹관리


## 기술 스택
### Language
   - Kotlin
### UI
   - Navigation Drawer
   - ViewPager2
   - 등.. 기타 ui components
### DI
   - Koin
### Network/비동기 처리
   - RxJava
   - Retrofit2
### Data Source
   - Kakao Api
   - Naver Api
### DB
   - Room
